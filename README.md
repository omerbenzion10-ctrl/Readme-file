# 🔐 Cyber Cryptography Project - 10th Grade

פרויקט סיום בקורס סייבר (כיתה י'), הכולל מערכת לניהול משתמשים וממשק גרפי להצפנה, פענוח ופריצה של צפנים מקוריים בשיטות מתקדמות.

## 📝 תיאור הפרויקט
המערכת מציעה ממשק משתמש ידידותי (מבוסס Gradio) המאפשר לבצע פעולות על שני סוגי צפנים ייחודיים שפותחו ב-Python. הפרויקט כולל מערכת הרשמה והתחברות מאובטחת המשתמשת בהצפנת סיסמאות (bcrypt) ושמירה בבסיס נתונים מקומי (JSON).

---

## 🛠 הצפנים במערכת

### 1. צופן מפת קואורדינטות (Coordinates Map Cipher)
צופן זה מבוסס על מטריצה של 10x10 הכוללת תווים ניתנים להדפסה.
* **הצפנה:** כל תו בטקסט הופך לזוג קואורדינטות (x, y) על המפה.
* **טוויסט:** לאחר כל תו שמוצפן, הטבלה כולה "זזה" ימינה מספר פעמים בהתאם למפתח (Key), מה שהופך את ההצפנה לדינמית וקשה לפיצוח ידני.
* **פלט:** רשימה של טאפלים, לדוגמה: `[(1,2), (3,5)]`.

### 2. צופן פיבונאצ'י קיסר (Fibonacci Caesar Cipher)
גרסה משודרגת של צופן קיסר הקלאסי.
* **הצפנה:** לכל תו מתווסף ערך המפתח + האיבר התואם בסדרת פיבונאצ'י לפי מיקומו בטקסט.
* **תכונות:** הצופן שומר על אותיות גדולות/קטנות ואינו משנה סימני פיסוק (אך הם עדיין נספרים לצורך קידום סדרת הפיבונאצ'י).
